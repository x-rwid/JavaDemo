## 🚀 Getting Started

Follow these steps to run any of the mini-projects on your local environment:

### Clone the Repository
```bash
git clone https://github.com
cd JavaDemo
```

### Run via Command Line
1. Navigate into the specific project directory:
   ```bash
   cd 01-car-models
   ```
2. Compile the source Java file:
   ```bash
   javac Main.java
   ```
3. Execute the compiled bytecode:
   ```bash
   java Main
   ```

### Run via IDE (IntelliJ IDEA / any)
1. Open your chosen IDE.
2. Choose **Open Project** and select the root directory of this repository.
3. The IDE will automatically index the sub-folders as packages or individual modules.
4. Open the `Main.java` file inside any project and click the green **Run** button.
